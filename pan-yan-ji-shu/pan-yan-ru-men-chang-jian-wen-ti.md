# 攀岩入门常见问题

## 攀岩小白

攀岩次数 < 5次，攀石 v0 - v1，顶绳5.8 - 5.9

| 心理         | 身体素质      | 技巧                     |
| ---------- | --------- | ---------------------- |
| 恐高、害怕坠落    | 手指酸痛，手皮脱落 | 不知道如何安全跳落（撞墙，身体前倾，手撑地） |
| 不敢转移重心     | 小臂酸胀      | 攀爬时手臂弯曲                |
| 不敢在岩壁上发力   | 没爬多久就精疲力尽 | 过度依赖上肢力量               |
| 不好意思在别人面前爬 |           |                        |



## 初来乍到

攀石 v2-v3, 顶绳 5.10a-b

| 心理         | 身体素质      | 技巧                     |
| ---------- | --------- | ---------------------- |
| 畏惧动态发力     | 手指酸痛，手皮脱落 | 不知道如何安全跳落（撞墙，身体前倾，手撑地） |
| 不信任脚点      | 小臂酸胀      | 攀爬时手臂弯曲                |
|            | 没爬多久就精疲力尽 | 过度依赖上肢力量               |
| 不好意思在别人面前爬 |           |                        |
